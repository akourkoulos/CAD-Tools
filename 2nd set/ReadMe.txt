## ΕΙΣΑΓΩΓΗ

Στην υλοποίηση αυτή προσομοιώνεται μία μόνο πύλη, έως τρεις εισόδους, χρησιμοποιώντας 
graph coloring ώστε να καθορίστεί η λειτουργικότητα της ψηφιακής αυτής πύλης, βρίσκοντας 
και βραχυκυκλώματα ή ασύνδετες εξόδους (εαν κάτι τέτοιο συμβαίνει), χωρίς όμως να μας απασχολούν εκπεφρασμένα οι 
τρικατάστατες έξοδοι ή συνδεσμολογία διακοπτών CMOS (CMOS switches).
